ACM Unfolder — Guide

Workflow

1. Load a Model
Use the "Load JSON" button to upload a model file exported from Blender, or select from the Models dropdown, or click "Demo" for a sample.

2. Adjust Dimensions
Set the X (width), Y (depth), and Z (height) values. The 3D model updates in real time.

3. Configure Unfolding
• Seed face: Which face to start unfolding from (usually a central or large face)
• Flat angle: Threshold (degrees) for detecting flat folds (angles close to 0° or 180°)

4. Click Unfold
The 2D flat pattern appears on the right. Red lines = cuts, blue lines = folds with angle labels.

5. Download SVG
Export the 2D pattern as SVG (compatible with Inkscape, CorelDRAW, Adobe Illustrator).

JSON Model Format

{
  "name": "My Model",
  "unit": "mm",
  "vertices": [[x,y,z], [x,y,z], ...],
  "faces": [[v0,v1,v2,v3], ...],
  "seam_edges": [[v0,v1], ...]
}

Edge Types in 2D Pattern

  • Red lines: Cut edges (sheet separates here)
  • Blue lines: Fold/bend edges with dihedral angle
  • Gray dashed: Seam edges (internal folding)

Fold Direction Labels

On blue fold lines, the angle is labeled. Direction:
• VALLEY: Concave/inward fold
• MOUNTAIN: Convex/outward fold
• FLAT: Minimal angle (<threshold or >180°−threshold)

Tips

  • Ensure face normals point outward in Blender (Edit Mode → Mesh → Normals → Recalculate Outside)
  • Mark seam edges in Blender with U → Mark Seam
  • For complex models, adjust the seed face if you see overlaps in the 2D pattern
  • Use the dimension sliders to scale the model before unfolding

Rate Limiting

Free tier users have 5 unfolds per day. The remaining count is displayed in the header. Counter resets at midnight UTC.
